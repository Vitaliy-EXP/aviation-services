# Авиационные услуги - тестовое задание

Простое приложение для выбора дополнительных услуг для авиапассажиров.

## Функционал

1. Просмотр списка услуг
2. Добавление услуг в заказ
3. Просмотр выбранных услуг и общей суммы
4. Оформление заказа (имитация)

## Технологии

- React 18
- TypeScript
- CSS Modules

## Установка и запуск

```bash
npm install
npm start
